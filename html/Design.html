Common Information
Header

I use header tag to contain header part of my page which contain a click-able logo which will go to my home page when clicked.

Navigation

For the navigation I combine all hyperlink corresponding to Home, Now Showing and Ticket page in nav tag which is a container for navigation buttons.

Footer

In the footer, I add text which tell about the copyright information in a p tag and also add the hyperlink to home and design page.

Home Page
Featured Video

I embed the video by providing two-source with two different filetypes using two source tags and alternative text in a p tag inside a video tag and add a line to distinguish the video with now showing content.

Now Showing and Coming Soon

I make a subdivision by using div tag for now showing and coming soon category and add the thumbnail for each movie using img tag and add the description below the thumbnail using p tag.

Now Showing Page
Cinema Location Selection

I use select tag inside a form tag to make the selection and asign a value which consisting the name of the location.

Time and House Selection

I use one form for each movie so when the "Buy Ticket" (submit) button is clicked, the only information that submitted only the one that corresponds to the movie.

Cinema Location

I use header (h3) tag to show the name of the location and below that I give detail addres using p tag

Ticket Page
Ticket Details

I use header (h3) tag to show all the details of the ticket.

Seat Selection

I use p tag to display the "screen" text and table tag to create a table of the seat arrangement; the use of the border is to visualize the table so it will more represent the seating arrangement; however, it will be later deleted and configured using CSS.

Seat confirmation

Inside a form tag, I use a fieldset and a legend to show what the section means and I use p tag to display the seat that will be booked and also there is a submit button to confirm the seating arrangement which when clicked the page will direct to Print page.

Print Page
Ticket Information

I use a label and textbox inside a form tag to show the ticket information; I also make the text readonly because this form tag shows the information, not asking the user to input some information.

Design Sketch Diagram
Style Sheets and Groups of Styles

There is one stylesheets to set overall theme which is black and white, fonts, header and footer. Also, every page has one group of style that used to configure the design for each corresponding page.

CSS3

I use box-shadow property in stylesheet "Header Footer.css" in line 28, to make a shadow in my navigation panel. I also use border-radius in stylegroup "ticket.html" line 40 and "now_showing.html" line 48,58,68,78 to make my button/input/selection has a round corner.

Event Handler
